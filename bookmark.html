THEY DON’T WANT YOU TO READ THIS is a bookmarklet that recreates the experience of witnessing a web page taken down by whoever has the power to control information. The result is a visually scrambled layout. The first effect you get when you run the bookmarklet is erasing all the text on the page. 2 minutes later, the page starts to be deconstructed.
